Staying focused

I've found that accessing my Facebook newsfeed on mobile is sufficient, and that I don't also need it on desktop. The best solution I've found:

  • Install the Stylus extension for Chrome

  • Create a new script applying to 'URLs on the domain: facebook.com': #contentArea > #stream_pagelet { display: none }
